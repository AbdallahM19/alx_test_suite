## How to use

There are 5 folders 1 per task

```
cd <Task Folder>
```

then

```
cp <your compiled shell> .
```

then

```
./checker.bash ./hsh <test file> 
```

example

```
cd 1.\ Simple\ shell 0.1
cp ~/simple_shell/hsh .
./checker.bash ./hsh bin_ls_3_times.bash
```

## OK means you passed anything else is a fail
